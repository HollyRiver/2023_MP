모든 개발(데이터과학 관련 코드)

> 리눅스
>> 코드스페이스를 이용하여 리눅스 환경을 경험해보자.

- 장점:

슈퍼장점: 자동으로 깃헙연결 되어있음 (요즘 깃험쓰기 짜증나서 이게 장점임)
로칼컴퓨터대비 장점: 내 컴퓨터 깨끗하게 쓸 수 있음 + 내 맘대로 무한 리눅스서버 사용가능
코랩/kaggle 대비 장점: 한번 정리된 셋팅이 영구적으로 적용됨. (autogluon 맨날 안깔아도 된다)
- 단점:

컴터가 별로임, 사용시간 제한있음 (월 100시간?)
단점을 극복하려면 결재해야함



- 초기설정: 복붙설정하기(파일에서 코드 복사하고 콘솔에서 현재 위치(파란 글씨) 누르기, 주피터 extension 설치 > 확장에서 깔 수 있는 것은 다 깔아둔다.



ls : 탐색기 파일의 이름을 보여준다. list의 약자?
ls -a : 숨긴 파일까지 알려준다.
pwd :  현재 디렉토리를 알려준다.
cd directory : 지정된 디렉토리로 이동
cd .. : 상위 디렉토리로 이동
cd . : 현재 디렉토리
cd .. 을 통해 최상위 디렉토리로 이동하면 파일 확인 가능

> cd 바리에이션

- ls ./notebooks/	: 현재 폴더(.)안의 어떤 폴더에 무슨 파일이 있는지
- cd /	: 최상위 디렉토리로 이동

mkdir 폴더명/	: 폴더 만들기
rm -rf 폴더명/	: 폴더 삭제
rm 파일병	: 파일 삭제

cp 파일명 경로 : 파일을 경로에 복사한다.
mv 파일명 경로 : 파일을 경로로 이동시킨다.

unzip

"!" 를 코드 앞에 붙이면 외부 콘솔에서 입력한 것과 동일하게 작동한다.
!ls : 현재 파일의 위치
!pwd : 현재 디렉토리 소환


kaggle의 setting에 있는 API를 다운받아 코드스페이스로 이동
> 다운로드 받으려면 직접 홈페이지 들어가서 다운도르 해야 했는데, 그걸 간단하게 하도록 하자.

pip install kaggle

cp kaggle.json /home/codespace/.kaggle	## kaggle.json을 해당 디렉토리로 이동시켜야 정상작동

kaggle competitions download -c titanic	## 파일을 현재 디렉토리에 저장한다.

unzip titanic.zip	## 파일을 압축해제한다.

rm *	## 모든 파일 삭제

코드스페이스에서 터미널도 새로 만들어낼 수 있음.
